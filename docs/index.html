AnyCSV: Parse ANY conceivable CSV string

Use this library to parse ANY CSV string, or whole file full of them.

The single public method, Parse, processes a single string. To process a whole file, wrap a standard StreamReader loop around an instance that has its properties set as needed.

This library is exposed to COM, so that it can be used to split strings in ways that no native Excel function can easily duplicate.

NuGet: This package is available on the public NuGet repository. Its ID is WizardWrx.AnyCSV. The NuGet package contains only the assembly and its debugging data base (.PDB) file. The assembly is signed with a strong name, so that it can be installed into a Global Assembly Cache. The public key token is 39f6551a21db420a.

Source Code: The source code is available on GitHub at https://github.com/txwizard/AnyCSV , and the license is a standard three-clause BSD license, as is the NuGet package.
